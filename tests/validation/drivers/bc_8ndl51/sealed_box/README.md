# Sealed Box Validation - BC 8NDL51

## Configuration

**Driver:** BC 8NDL51-8
**Alignment:** Qtc = 0.707 (Butterworth, maximally flat)
**Box Volume:** Vb = 31.65 L
**System Resonance:** Fc = 86.1 Hz
**System Q:** Qtc = 0.707

## Files

- `input_qtc0.707.txt` - Hornresp input file (generated by viberesp)
- `sim.txt` - Hornresp simulation results (**YOU NEED TO GENERATE THIS**)

## How to Generate Hornresp Data

### Step 1: Open Hornresp and Import

1. Launch Hornresp
2. File → Open
3. Select `input_qtc0.707.txt`

### Step 2: Verify Parameters

You should see:
- **Driver Parameters:**
  - Sd = 220 cm²
  - Bl = 7.3 T·m
  - Mmd = 26.29 g
  - Cms = 1.50E-04
  - Rms = 2.44
  - Re = 2.60 Ω
  - Le = 0.15 mH

- **Enclosure:**
  - Vrc = 31.65 L (sealed box)
  - Lrc = 31.63 cm (chamber depth)
  - S1 = 0, L45 = 0 (infinite baffle front)

### Step 3: Run Simulation

1. Click "Calculate" or press Ctrl+L
2. Accept defaults (frequency range 10-20000 Hz, 2.83V input)

### Step 4: Export Results

1. File → Save
2. Select "Export _sim.txt" format
3. Save as `sim.txt` in this directory

### Step 5: Run Validation Tests

```bash
PYTHONPATH=src pytest tests/validation/test_sealed_box.py -v
```

## Expected Results

Based on infinite baffle validation, expect:
- **Electrical impedance magnitude:** <5% above 200 Hz
- **Electrical impedance phase:** <10° above 200 Hz
- **SPL:** <6 dB max, <4.5 dB RMS
- **System parameters (Fc, Qtc):** <0.5 Hz, <0.02 tolerance
